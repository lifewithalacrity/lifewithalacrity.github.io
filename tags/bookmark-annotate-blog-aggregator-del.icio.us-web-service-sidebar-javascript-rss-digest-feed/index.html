New Sidebar: Recent Bookmarks

April 8 2005 - 600 Words
If you read my blog through an aggregator, you may not have noticed my new sidebar "Recent Bookmarks". It is a list of web pages that I've found interesting enough to annotate using the del.icio.us service. It is useful if you want to have an insight into what future blog entries I'm working on, as links will often show up there before my actual blog posting is out. You alternatively can view my last 10 del.icio.us postings, or subscribe to my del.icio.us RSS feed.
